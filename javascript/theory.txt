7. for...of vs for...in
Both loops are used for iteration, but they are used in different situations.

for...of: 
Yeh kisi iterable object (jaise Array, String, Map) ki values nikalne ke liye use hota hai.

for...in:
Yeh kisi object ki keys (properties) ya array ke index nikalne ke liye use hota hai.

9. Event Delegation:
Event Delegation is a technique where a parent element handles events for its child elements using event bubbling.

10. Local Storage

11. Event Bubbling
Event Bubbling, JavaScript mein ek event propagation ka tarika hai. Jab aap kisi nested element (ek element ke andar dusra element) par click karte hain, tab event sabse andar wale element se shuru hokar upar parents ki taraf 'bubble' (flow) karta hai.
